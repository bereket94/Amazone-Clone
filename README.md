# Amazon Clone

A responsive front-end clone of Amazon's homepage, built with HTML, CSS, and JavaScript. This project demonstrates modern UI design principles including product sliders, grid layouts, and interactive navigation.

## Features

- **Responsive Navigation Bar** with search, location selector, language picker, account menu, and cart
- **Auto-playing Image Slider** for header promotions
- **Product Card Grids** with hover effects and deal badges
- **Multiple Product Sections** including:
  - Category boxes with images
  - Grid-based product showcases
  - Horizontal product sliders
  - "Deals Under $25" section with pricing and discount tags
- **Complete Footer** with multi-column links and legal information
- **Mobile-responsive** design (via CSS media queries)

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- JavaScript (slider functionality)
- Google Material Icons
- Font Awesome
